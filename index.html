Setting up

Things you need to get started:

  • arduino uno
  • data logging shield (or equivalent board)
  • ds18b20 temperature sensor
  • light dependent resistor (ldr)
  • Some sort of battery
  • Wire
  • Heat shrink
  • Water/Tamperproof container

Setting up the arduino

To start I'm using an arduino uno with a string of six light dependent resistors to measure light intensity, and a matching set of six dallas onewire digital temperature sensors (ds18b20).

To get started taking readings from the ldr I referred to instructions on this page. Using these instructions, I was able to start taking readings from a single ldr within about 2 minutes.

I used a similar set of instructions for the ds18b20 temperature sensor, using the instructions here. Note that it is necessary to install two additional libraries in order to use the ds18b20 with the arduino: the Dallas Temperature Control Library and the OneWire library.
